Hidden patterns in a museum collection

A museum search is a funnel: every filter narrows a list toward the one object you already wanted. These four views do the opposite — they show the shape of the collection first, so its structure, its gaps, and what it makes visible become the subject.

All four representations draw on the same public-domain working subset of 30,000 objects prepared from the Met Open Access dataset. Each one asks a single question: what does the collection reveal that a table of records keeps hidden?
