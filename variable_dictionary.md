# Variable Dictionary

| Variable              | Description                                                                |
| --------------------- | -------------------------------------------------------------------------- |
| `object_id`           | Unique identifier of the object in The Met collection.                     |
| `object_number`       | Museum object number.                                                      |
| `is_highlight`        | Indicates whether the object is marked as a highlight by The Met.          |
| `is_timeline_work`    | Indicates whether the object appears in The Met timeline resources.        |
| `is_public_domain`    | Indicates whether the object is in the public domain.                      |
| `object_name`         | General object type or object name.                                        |
| `title`               | Title of the artwork or collection object.                                 |
| `culture`             | Cultural attribution of the object, when available.                        |
| `period`              | Historical or artistic period, when available.                             |
| `artist_name`         | Displayed artist name.                                                     |
| `artist_nationality`  | Nationality of the artist, when available.                                 |
| `artist_gender`       | Gender of the artist, when available.                                      |
| `object_date`         | Original textual date description.                                         |
| `object_begin_date`   | Numeric beginning date of the object.                                      |
| `object_end_date`     | Numeric ending date of the object.                                         |
| `medium`              | Original material or technique description.                                |
| `dimensions`          | Physical dimensions of the object.                                         |
| `classification`      | Object classification used by The Met.                                     |
| `department`          | Museum department responsible for the object.                              |
| `credit_line`         | Acquisition or credit information.                                         |
| `geography_type`      | Type of geographic attribution.                                            |
| `city`                | City associated with the object, when available.                           |
| `state`               | State associated with the object, when available.                          |
| `county`              | County associated with the object, when available.                         |
| `country`             | Country associated with the object, when available.                        |
| `region`              | Region associated with the object, when available.                         |
| `subregion`           | Subregion associated with the object, when available.                      |
| `link_resource`       | URL of the object page on The Met website.                                 |
| `object_wikidata_url` | Wikidata URL associated with the object, when available.                   |
| `tags`                | Subject tags associated with the object.                                   |
| `object_mid_year`     | Calculated average year between `object_begin_date` and `object_end_date`. |
| `time_period_group`   | Simplified historical time-period category.                                |
| `medium_group`        | Simplified material or medium category.                                    |
| `has_geography_info`  | Indicates whether the record has country, region, or culture information.  |
| `visibility_group`    | Grouped visibility status based on highlight and timeline fields.          |
